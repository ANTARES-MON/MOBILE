# Mobile Applications

Collection of Android apps with secure configuration.

## Setup
1. Run: `./setup-projects.sh`
2. Edit each `project/local.properties` with your API keys
3. NEVER commit `local.properties` files

## Projects
- MapsApp: Google Maps integration
- GPSLocation: Location services
- WIFI: WiFi scanning
- BLUETOOTH: Bluetooth management
- SMS: Messaging app
- CONVERSION: Unit converter
- PERIMETRE: Geometry calculator
- CALC: Calculator
- LISTVIEW: List demonstrations
- MENU: Menu examples
- SqlLiteF: Database app
- AllApps: Launcher application
